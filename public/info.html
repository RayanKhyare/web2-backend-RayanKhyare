Documentation REST Api Rayan Khyare

/ = Documentation


GET /users =Show all users

GET /users/:id = Show one specific user

POST /users =Post a user

POST /users/login = Verify is user exists

DEL /users/:id = Delete an user

PUT /users/:id = Update email or password of user


GET /games =Show all saved games

GET /games/:id =Show games saved by a specific user

GET /games/bookmarks/:id

POST /games = Post a game

DEL /games/:id = Delete a game
